About project

This project aims to analyse the emotion from the uploaded image/webcam images of a person. Machine Learning techniques, Deep Learning models, and Neural Network algorithms can be used for emotion recognition. Here,we have used convolutional neural networks (CNNs) to identify the facial emotions of the person and then classify to produce the result based on the classification.
